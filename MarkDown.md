## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочками, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать, например, вот так:
1. Первый пункт
2. Второй пункт
